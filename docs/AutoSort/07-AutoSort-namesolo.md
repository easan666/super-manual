# <span class="h1-icon"><img src="/img/NameSolo.webp" alt="SelectSolo文档"></span>Name Solo（同名独显）

## 说明

根据名称过滤相关对象，单击可将所选独显（可用于灯光独显）

::: info 注意
可以在插件设置中添加关键词，会根据名称过滤对象。示例：`区域光,穹顶光,天空,Sky`, 用英文逗号`,`分隔；
:::
