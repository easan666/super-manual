# <span class="h1-icon"><img src="/auto_sort/Set_Obj_Random_Color.webp" alt="SelectSolo文档"></span>Set Obj Random Color（对象随机色）

## 说明

选择对象单击可以设置对象随机颜色, Alt或者Ctrl+单击是取消对象颜色

<br />

::: info 注意
此功能默认只对当前所选对象设置，如果需要设置所选对象的子级，需要右键`选择子级`，然后点击
:::
