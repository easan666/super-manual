# <span class="h1-icon"><img src="/img/generate_curve.webp" alt="keytool文档"></span>曲线生成

## 简介

- 选择对象，点击可以根据对象的位置生成对应的位置曲线
- 建议至少选择三个对象，如果曲线不平滑可以调整`点插值的数量`
- 一般是用在获取多个摄象机的位置生成曲线
