# <span class="h1-icon"><img src="/img/split_pr.webp" alt="keytool文档"></span>拆分P/R

## 简介

- 选择对象，点击可以分离对象的位置和旋转动画，并且设置对应的null为父级
- 一般用于已经K好动画的摄像机，点击就可以分离，方便后期调整
