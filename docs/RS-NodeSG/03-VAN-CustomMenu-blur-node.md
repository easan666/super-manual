# <span class="h1-icon"><img src="/img/SG-Blur Node.webp" alt="Custom Icon"></span>Blur Node（模糊节点）

<br/>

---

### 简介

- 可以为贴图节点添加模糊节点设置模糊效果，支持批量添加
- 可以全选节点，会自动过滤，只为贴图节点添加模糊节点

> Blur Node（模糊节点）是OSL，首次添加会卡顿2秒左右（加载osl文件）
