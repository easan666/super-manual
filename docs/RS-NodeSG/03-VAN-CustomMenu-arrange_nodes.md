# <span class="h1-icon"><img src="/img/SG-Arrange nodes.webp" alt="Custom Icon"></span>Arrange nodes（排列节点）

<br/>

---

### 简介
- 主要用于重叠的节点，选中后点击会自动排列
- Vertically：垂直排列
- Horizontally：水平排列
- Align top，Align bottom，Align left，Align right：上下左右对齐

<br/>

### 使用
选中所需节点，然后点击即可
