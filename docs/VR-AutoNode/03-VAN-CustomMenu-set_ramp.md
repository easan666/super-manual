# <span class="h1-icon"><img src="/img/VR-CustomMenu.webp" alt="Custom Icon"></span>Ramp（渐变映射）


<br/>

- 用于调用导入的节点资产Color Ramp，用于控制黑白贴图明暗
- 插件自定义id： `set_ramp`


> 首次使用需要需要导入VR-AutoNode.zip文件

<br/>

### 选择节点插入
* 选中节点，然后在弹出菜单中选择Ramp
